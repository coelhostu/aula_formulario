# 📋 Aula de Formulários HTML

Este projeto é um guia educacional prático projetado para aulas de desenvolvimento web sobre **Formulários HTML**. Ele ensina como estruturar formulários acessíveis, semânticos e válidos usando recursos nativos do HTML5.

🔗 **Link para o Guia e Laboratório:** [https://coelhostu.github.io/aula_formulario/](https://coelhostu.github.io/aula_formulario/)

## 🌟 Recursos

1. **Guia Passo a Passo**: Capítulos interativos abordando tags, inputs, validação nativa e boas práticas.
2. **Laboratório Interativo (Modo Visual)**: Um playground de código onde alunos e professores podem editar HTML de formulários, ver o resultado em tempo real, testar submissões (simulando a recepção dos dados pelo servidor) e auditar a acessibilidade do código.

## 🚀 Como Executar Localmente

### 1. Instalar dependências
```bash
npx pnpm install
```

### 2. Iniciar o projeto
```bash
PORT=8080 BASE_PATH=/ npx pnpm --filter @workspace/html-forms-guide dev
```
Abra seu navegador em [http://localhost:8080](http://localhost:8080) para acessar a página.
